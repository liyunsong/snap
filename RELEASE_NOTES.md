# Snap v1.0.2

## 修复

- **首次打开被拦截**：macOS 15 对未公证的网上下载应用会弹出「未打开 Snap.app / Apple 无法验证…」，仅显示「完成」。
- DMG 内新增 **「首次打开 Snap.command」**：先把 Snap 拖到应用程序，再双击该脚本即可清除隔离属性并启动。
- 文档补充 Sequoia「系统设置 → 隐私与安全性 → 仍要打开」与终端 `xattr` 两种绕过方式。

> 请使用 **v1.0.2** 的 `Snap-1.0.2.dmg`。v1.0.0 包损坏；v1.0.1 可装但首次打开仍会被 Gatekeeper 拦截（可用下方方法绕过）。

## 📋 系统要求

- macOS 15.0 (Sequoia) 或更高版本
- Apple Silicon (arm64) Mac

## 📦 安装方法

1. 下载 `Snap-1.0.2.dmg`，双击打开
2. 将 **Snap** 拖到 **Applications（应用程序）**
3. **若出现「未打开 Snap.app」/「Apple 无法验证」**（只有「完成」按钮），任选其一：

   **方法 A（推荐）**：在 DMG 窗口双击 **「首次打开 Snap.command」**，按提示在终端允许运行

   **方法 B（系统设置）**：
   1. 点「完成」
   2. 打开 **系统设置 → 隐私与安全性**
   3. 滚到「安全性」，找到 Snap 被拦截的提示，点 **仍要打开**
   4. 再确认一次打开

   **方法 C（终端）**：
   ```bash
   xattr -cr /Applications/Snap.app
   open /Applications/Snap.app
   ```

4. 授予「屏幕录制」权限后即可使用

> 说明：本应用为开源免费分发，未购买 Apple 开发者证书做公证，因此 macOS 会拦截一次。允许后即可正常使用，应用本身并未损坏。

## 🎉 功能概览

- 区域截图 (`⌘⇧A`) / 全屏截图 (`⌘⇧S`)
- 矩形、圆形、箭头、直线、画笔、文字、马赛克、模糊
- 复制到剪贴板 / 保存到桌面
- 菜单栏常驻，可自定义快捷键
