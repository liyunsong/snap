# Snap 使用说明 - M1 Pro 专用版

专为 Apple Silicon (M1 Pro) macOS 26 优化的截图应用。

## 🚀 快速开始（3步）

### 1️⃣ 构建应用

打开终端，进入项目目录，运行：

```bash
./build-local.sh
```

### 2️⃣ 安装应用

```bash
sudo cp -r .build/release/Snap.app /Applications/
```

### 3️⃣ 运行应用

```bash
open /Applications/Snap.app
```

完成！现在你会在菜单栏看到 📷 图标。

## ⚙️ 首次使用设置

### 授予屏幕录制权限

首次运行会弹出权限请求：

1. 点击 **打开系统设置**
2. 或手动打开：**系统设置** → **隐私与安全性** → **屏幕录制**
3. 找到并勾选 **Snap**
4. 重启 Snap 应用

### 允许打开应用

如果看到"无法验证开发者"提示：

**方法 1**：右键打开
- 右键点击 `/Applications/Snap.app`
- 选择 **打开**
- 在弹出对话框点击 **打开**

**方法 2**：使用终端
```bash
xattr -cr /Applications/Snap.app
open /Applications/Snap.app
```

## 📸 如何使用

### 截图方式

#### 区域截图 ⌘⇧A
1. 按下 `⌘ + ⇧ + A`
2. 鼠标拖拽选择区域
3. 松开鼠标完成选择
4. 自动打开编辑器

#### 全屏截图 ⌘⇧S
1. 按下 `⌘ + ⇧ + S`
2. 自动捕获整个屏幕
3. 打开编辑器

#### 从菜单栏
1. 点击菜单栏的 📷 图标
2. 选择 **区域截图** 或 **全屏截图**

### 编辑工具

截图后会自动打开编辑器，右侧有工具面板：

#### 绘图工具
- 🔲 **矩形** - 绘制矩形框
- ⭕ **圆形** - 绘制圆形/椭圆
- ➡️ **箭头** - 绘制带箭头的指示线
- 📏 **直线** - 绘制直线
- ✏️ **画笔** - 自由绘制路径
- 🔤 **文字** - 添加文字标注

#### 效果工具
- 🔳 **马赛克** - 像素化（隐私保护）
- 🌫️ **模糊** - 高斯模糊效果

#### 编辑功能
- 🎨 **颜色** - 点击色块选择颜色（10+ 颜色）
- 📐 **粗细** - 调节线宽（1-20px）
- ↩️ **撤销** - 撤销上一步操作
- ↪️ **重做** - 恢复撤销的操作
- 📋 **复制** - 复制到剪贴板
- 💾 **保存** - 保存到桌面（PNG格式）
- ❌ **关闭** - 关闭编辑器

### 保存截图

#### 复制到剪贴板
1. 编辑完成后点击 **复制** 按钮
2. 在任意应用中按 `⌘ + V` 粘贴

#### 保存到文件
1. 点击 **保存** 按钮
2. 文件自动保存到桌面
3. 文件名格式：`Screenshot-2026-07-15-06-43-21.png`

## ⚙️ 自定义设置

### 修改快捷键

1. 点击菜单栏 📷 图标
2. 选择 **设置**
3. 点击快捷键录制器
4. 按下你想要的快捷键组合
5. 新快捷键立即生效

默认快捷键：
- 区域截图：`⌘ + ⇧ + A`
- 全屏截图：`⌘ + ⇧ + S`

### 开机自动启动

1. **系统设置** → **通用** → **登录项**
2. 点击 **+** 号
3. 选择 `/Applications/Snap.app`
4. 下次开机会自动启动

## 🐛 常见问题

### Q: 快捷键不工作
**A**: 检查是否与系统快捷键冲突：
- 系统设置 → 键盘 → 键盘快捷键
- 如有冲突，在 Snap 设置中更换快捷键

### Q: 截图是黑屏
**A**: 没有授予屏幕录制权限：
- 系统设置 → 隐私与安全性 → 屏幕录制
- 勾选 Snap
- 重启应用

### Q: 应用闪退
**A**: 查看系统日志：
```bash
log show --predicate 'process == "Snap"' --last 5m
```

### Q: 内存占用高
**A**: 编辑完成后点击"关闭"释放资源。
正常占用应该 < 50MB

### Q: 构建失败
**A**: 确保已安装 Xcode Command Line Tools：
```bash
xcode-select --install
```
